# 脑暴方法参考

## 自由联想

**适合场景**：通用，无特定方向
**步骤**：
1. 给出核心词/主题
2. 用户自由说出联想到的 5-10 个词
3. 对每个联想词再次联想（二级联想）
4. 整理联想网络，寻找连接点

```
核心词: "物流"
  → 快递 → 包裹 → 驿站 → 扫码 → 签收
  → 运输 → 卡车 → 路线 → 导航 → 调度
  → 时效 → 加急 → 延误 → 投诉 → 赔偿
```

## SCAMPER 法

**适合场景**：已有产品/方案的改进优化

| 字母 | 含义 | 引导问题 |
|------|------|----------|
| **S** | Substitute（替代） | 有什么可以替换？用其他材料/流程/人？ |
| **C** | Combine（组合） | 可以和什么组合？功能合并？ |
| **A** | Adapt（改编） | 有什么可以借鉴？其他行业的做法？ |
| **M** | Modify（修改） | 放大/缩小/改变形状/颜色？ |
| **P** | Put to other uses（他用） | 还有别的用途吗？换个场景？ |
| **E** | Eliminate（去除） | 可以去掉什么？简化什么？ |
| **R** | Reverse（逆向） | 反过来做？顺序调换？ |

## 六顶思考帽

**适合场景**：复杂问题全面分析，决策评估

| 帽子 | 颜色 | 思考方向 | 引导 |
|------|------|----------|------|
| 🟦 蓝帽 | 蓝色 | 组织思维，控制流程 | 我们今天要讨论什么？目标是什么？ |
| 🟥 红帽 | 红色 | 直觉、情感、预感 | 你觉得怎么样？直觉告诉你什么？ |
| ⬛ 黑帽 | 黑色 | 风险、问题、困难 | 这个方案有什么风险？哪些可能出错？ |
| 🟨 黄帽 | 黄色 | 价值、利益、优点 | 这个方案有什么好处？价值在哪里？ |
| 🟩 绿帽 | 绿色 | 创意、新点子、可能 | 有什么新思路？有没有其他办法？ |
| ⬜ 白帽 | 白色 | 事实、数据、信息 | 我们有哪些数据支持？还缺什么信息？ |

**执行流程**：每顶帽子逐一佩戴，每轮专注一个角度

## 头脑书写 (Brainwriting)

**适合场景**：团队协作，避免群体压力

**步骤**：
1. 每人一张纸（或 Agent 逐轮引导）
2. 每轮 3-5 分钟写出 3 个想法
3. 传给下一个人，在其想法基础上继续发想
4. 重复 3-5 轮
5. 收集所有纸张，整理分类

**单人版（Agent 引导）**：
```
第一轮：请写出 3 个和主题相关的想法
→ [用户写出 3 个]

第二轮：针对你的想法 1，怎样可以更好？
→ [用户改进]

第三轮：针对你的想法 2，换个角度想想
→ [用户换角度]
```

## 思维导图

**适合场景**：主题分解，结构探索

**步骤**：
1. 中心写主题
2. 一级分支：核心维度
3. 二级分支：每个维度的具体想法
4. 三级分支：进一步细化

```
                 ┌── 功能A
          ┌── 功能 ── 功能B
          │         └── 功能C
    ┌── 产品 ── 设计 ── 配色
    │         │         └── 交互
    │         └── 体验 ── 流程
    │                   └── 反馈
  [主题]
    │         ┌── 学生
    │         ├── 上班族
    └── 用户 ── 老年人
              └── 企业
```

## 反向脑暴

**适合场景**：找问题、找风险

**步骤**：
1. 不是问"如何解决问题"，而是问"如何制造问题"
2. 列出所有可能让事情变糟的方式
3. 每个"坏事"反转 → 得到解决方案

```
正向：如何提高用户留存？
反向：如何让用户快速流失？
  → 复杂注册流程 → 解决方案：简化注册
  → 频繁推送广告 → 解决方案：个性化推送
  → 加载很慢     → 解决方案：性能优化
```

## 方法选择指南

| 议题类型 | 推荐方法 | 原因 |
|----------|----------|------|
| 新产品构思 | 自由联想 → 思维导图 | 先发散，再结构化 |
| 现有产品优化 | SCAMPER | 系统化改进框架 |
| 问题分析/决策 | 六顶思考帽 | 多维度全面分析 |
| 风险排查 | 反向脑暴 | 快速发现盲区 |
| 创意深度挖掘 | 头脑书写 | 逐轮深化想法 |
| 战略规划 | 思维导图 + 六顶思考帽 | 结构 + 全面 |

## 发散 vs 收敛 引导话术

### 发散阶段
- "不用想好不好，先把想法说出来"
- "如果没有任何限制，你会怎么做？"
- "还能再多一个吗？"
- "这个想法很有意思，顺着它还能想到什么？"

### 收敛阶段
- "现在我们来看看哪些想法最可行"
- "这些想法可以分成几类？"
- "如果只能选 3 个，你选哪 3 个？"
- "这个想法落地需要什么条件？"
