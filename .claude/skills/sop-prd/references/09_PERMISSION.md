## 5. 权限验证流程

### 5.1 权限检查流程

```mermaid
graph TD
  A[请求] --> B{是否有权限?}
  B -->|是| C[放行]
  B -->|否| D[返回403]
```

### 5.2 接口权限注解

```java
// 示例：仅管理员可访问
@RequiresRoles("admin")
@RequiresPermissions("system:user:delete")
@DeleteMapping("/users/{id}")
public void deleteUser() {}

// 示例：多角色任一可访问
@RequiresRoles(value = {"admin", "manager"}, logical = Logical.OR)
@RequiresPermissions("system:user:query")
@GetMapping("/users")
public List<User> listUsers() {}
```
