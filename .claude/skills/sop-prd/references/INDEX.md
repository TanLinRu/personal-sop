# PRD 文档模板索引

> PRD 文档体系 - 知识驱动 + 分层设计

## 文档层级

| 层级 | 文档 | 适用场景 | 配套附件数 |
|------|------|----------|------------|
| **精简版** | PRD 核心 | 快速迭代、内部项目 | 3-4 个 |
| **标准版** | BRD + PRD | 商业项目、对外汇报 | 6-7 个 |
| **完整版** | BRD + MRD + PRD | 大型项目、立项汇报 | 9+ 个 |

## 模板清单

| # | 模板 | 层级 | 说明 |

## 快速选择指南

### 精简版组合（推荐内部项目）
```
05_PRD_LITE + 06_FLOW + 08_DATA_DICT + 10_NFR
```

### 标准版组合（推荐商业项目）
```
01_BRD + 04_PRD_FULL + 06_FLOW + 07_SEQUENCE + 08_DATA_DICT + 09_PERMISSION + 10_NFR
```

### 完整版组合（推荐大型项目）
```
01_BRD + 02_MRD + 04_PRD_FULL + 06~13 全部
```

## 使用方式

在 sop-prd 执行过程中，通过 Step 1.5 文档类型选择后：

1. **自动引用**：根据选择的文档类型，自动引用对应模板
2. **手动下载**：从 references/ 目录获取模板文件
3. **组合输出**：生成 .sop/output/prd-{name}.md 主文档 + 配套附件

## 维护说明

- 每个模板独立版本管理
- 支持按需组合
- 可自定义模板后缀（如 `04_PRD_FULL_V2.md`）
- 模板中 `{{变量}}` 表示需填写的占位符
