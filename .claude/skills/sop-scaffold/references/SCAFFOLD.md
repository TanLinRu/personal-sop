# SCAFFOLD.md - 项目脚手架规范 (v2.1)

## 流程 (v2.1)

```
需求确认 → 需求调研 → PRD → 架构设计 → 架构审核 → 依赖查询 → 并行生成 → 验证 → 知识索引
```

## 步骤说明

| Step | 说明 | 关键输出 |
|------|------|----------|
| 1 | 需求确认 | 项目需求文档 |
| 2 | 需求调研 | 5领域 x 2次调研 |
| 3 | 生成 PRD | PRD 文档 |
| 4 | 架构设计 | 技术方案 |
| 5 | 架构审核 | 审核通过 |
| 6 | 依赖查询 | 避免冲突 |
| 7 | 并行生成 | 代码 |
| 8 | 验证 | 编译测试 |
| 9 | 知识索引 | 知识库 |
