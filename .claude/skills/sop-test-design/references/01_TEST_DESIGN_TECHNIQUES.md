# 测试设计技术参考

## 1. 等价类划分 (Equivalence Partitioning)

将输入数据分为若干等价类，从每个类中选取一个代表值作为测试用例。

### 有效等价类
符合规格说明的合理输入值。

### 无效等价类
不符合规格说明的不合理输入值。

### 示例：年龄输入（18-65岁）
| 等价类 | 输入值 | 预期结果 |
|--------|--------|----------|
| 有效 | 25 | 接受 |
| 无效（过小） | 10 | 拒绝 |
| 无效（过大） | 70 | 拒绝 |

---

## 2. 边界值分析 (Boundary Value Analysis)

测试边界值及其邻近值。

### 边界值规则
- 最小值、最小值-1
- 最大值、最大值+1
- 典型中间值

### 示例：密码长度（8-20字符）
| 边界 | 输入值 | 预期结果 |
|------|--------|----------|
| 最小值-1 | 7字符 | 拒绝 |
| 最小值 | 8字符 | 接受 |
| 最小值+1 | 9字符 | 接受 |
| 最大值-1 | 19字符 | 接受 |
| 最大值 | 20字符 | 接受 |
| 最大值+1 | 21字符 | 拒绝 |

---

## 3. 决策表 (Decision Table)

多条件组合的系统化测试方法。

### 结构
| 条件/动作 | 规则1 | 规则2 | 规则3 | 规则4 |
|-----------|-------|-------|-------|-------|
| 条件1 | Y | Y | N | N |
| 条件2 | Y | N | Y | N |
| 动作1 | X | X | - | - |
| 动作2 | - | - | X | - |

### 示例：订单折扣
| 条件/动作 | R1 | R2 | R3 | R4 |
|-----------|----|----|----|----|
| VIP会员 | Y | Y | N | N |
| 订单>100元 | Y | N | Y | N |
| 打8折 | X | - | - | - |
| 打9折 | - | X | - | - |
| 打95折 | - | - | X | - |
| 无折扣 | - | - | - | X |

---

## 4. 状态转换 (State Transition)

描述系统在不同状态下的行为。

### 状态转换图
```
[待支付] --支付成功--> [已支付]
[待支付] --取消--> [已取消]
[已支付] --发货--> [已发货]
[已发货] --签收--> [已完成]
[已支付] --退款--> [已退款]
```

### 测试用例
| 当前状态 | 事件 | 预期状态 | 测试用例 |
|----------|------|----------|----------|
| 待支付 | 支付成功 | 已支付 | TC-001 |
| 待支付 | 取消 | 已取消 | TC-002 |
| 已支付 | 发货 | 已发货 | TC-003 |
| 已支付 | 退款 | 已退款 | TC-004 |

---

## 5. 配对测试 (Pairwise Testing)

当有多个参数组合时，确保每两个参数的所有可能组合至少出现一次。

### 示例：3个参数
- 浏览器: Chrome, Firefox, Safari
- 操作系统: Windows, macOS, Linux
- 分辨率: 1920x1080, 1366x768

### 配对组合（9个用例覆盖所有两两组合）
| 用例 | 浏览器 | 操作系统 | 分辨率 |
|------|--------|----------|--------|
| 1 | Chrome | Windows | 1920x1080 |
| 2 | Chrome | macOS | 1366x768 |
| 3 | Chrome | Linux | 1920x1080 |
| 4 | Firefox | Windows | 1366x768 |
| 5 | Firefox | macOS | 1920x1080 |
| 6 | Firefox | Linux | 1366x768 |
| 7 | Safari | Windows | 1920x1080 |
| 8 | Safari | macOS | 1366x768 |
| 9 | Safari | Linux | 1920x1080 |

---

## 技术选择指南

| 需求特征 | 推荐技术 | 原因 |
|----------|----------|------|
| 输入验证 | 等价类 + 边界值 | 覆盖有效/无效输入和边界 |
| 多条件业务规则 | 决策表 | 系统化覆盖所有条件组合 |
| 有状态的工作流 | 状态转换 | 覆盖所有状态-事件组合 |
| 多参数配置 | 配对测试 | 高效覆盖两两组合 |
| 复杂逻辑 | 因果图 | 分析输入-输出因果关系 |
